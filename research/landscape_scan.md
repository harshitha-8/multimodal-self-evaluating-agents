# Research Landscape Scan

## Cluster 1: Self-Reflective Multimodal Agents
Agents that evaluate and refine their own multimodal reasoning.
- **Reflexion** (Shinn et al., NeurIPS 2023) — verbal self-reflection for task improvement
- **Self-Refine** (Madaan et al., NeurIPS 2023) — iterative refinement with self-feedback
- **CRITIC** (Gou et al., ICLR 2024) — tool-interactive critique for LLM self-correction
- **Introspective Tips** (Chen et al., 2024) — teaching LLMs to introspect

## Cluster 2: Vision-Language-Action Models
Models connecting perception to action through language reasoning.
- **RT-2** (Brohan et al., 2023) — vision-language-action model for robotics
- **PaLM-E** (Driess et al., ICML 2023) — embodied multimodal language model
- **VIMA** (Jiang et al., ICML 2023) — multimodal prompting for robot manipulation
- **Octo** (Team et al., 2024) — open-source generalist robot policy

## Cluster 3: Tool-Augmented Reasoning
Agents that dynamically invoke external tools for reasoning.
- **Toolformer** (Schick et al., NeurIPS 2023) — LLMs that teach themselves tool use
- **MM-ReAct** (Yang et al., 2024) — multimodal ReAct for vision-language tasks
- **Visual ChatGPT** (Wu et al., 2023) — connecting ChatGPT with visual foundation models
- **HuggingGPT** (Shen et al., NeurIPS 2023) — solving AI tasks with ChatGPT and friends

## Cluster 4: Multimodal Retrieval-Augmented Generation
RAG systems extended to multimodal inputs.
- **MuRAG** (Chen et al., 2022) — multimodal retrieval-augmented generation
- **Self-RAG** (Asai et al., ICLR 2024) — learning to retrieve, generate, and critique
- **RA-CM3** (Yasunaga et al., NeurIPS 2023) — retrieval-augmented multimodal models
- **Wiki-LLaVA** (Caffagni et al., 2024) — retrieval-augmented VLM

## Cluster 5: Uncertainty-Aware Multimodal Systems
Systems that estimate and use uncertainty in multimodal reasoning.
- **Conformal Prediction for VLMs** (various, 2024) — calibrated prediction sets
- **LLM Calibration** (Kadavath et al., 2022) — language models know what they know
- **Semantic Uncertainty** (Kuhn et al., ICLR 2023) — uncertainty in NLG
- **Factuality in LLMs** (Min et al., 2023) — FActScore for factual precision

## Cluster 6: Agent Evaluation Benchmarks
Benchmarks for evaluating agent capabilities.
- **AgentBench** (Liu et al., ICLR 2024) — evaluating LLMs as agents
- **IDEA-Bench** (2024) — comprehensive agent evaluation
- **ToolBench** (Qin et al., 2024) — tool-use evaluation
- **MM-Vet** (Yu et al., 2024) — evaluating multimodal models

## Cluster 7: Self-Improving Models
Models that improve through self-generated feedback.
- **STaR** (Zelikman et al., NeurIPS 2022) — self-taught reasoner
- **ReST** (Gulcehre et al., 2023) — reinforced self-training
- **Self-Play** (Chen et al., 2024) — SPIN for LLM alignment
- **Quiet-STaR** (Zelikman et al., 2024) — learning to think before speaking

## Cluster 8: Process Supervision
Evaluating intermediate reasoning steps, not just outcomes.
- **Let's Verify Step by Step** (Lightman et al., ICLR 2024) — process reward models
- **Math-Shepherd** (Wang et al., 2024) — process supervision for math reasoning
- **OmegaPRM** (Luo et al., 2024) — automated process reward modeling
- **RLHF-V** (Yu et al., 2024) — aligning with human values via process feedback
